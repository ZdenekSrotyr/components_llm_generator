=============

Okta is a platform that helps organizations manage and secure user authentication and access to applications, devices, and data across various systems.

**Table of contents:**

[TOC]


Prerequisites
=============

Organization url and API token


Supported endpoints
===================
 
 - users
 - user_types
 - devices

Configuration
=============

## Okta Extractor configuration
 - Organization url (org_url) - [REQ] see Sample Configuration for example
 - API Token (#api_token) - [REQ] API Key to Authenticate the connection with Octa.


## Okta Extractor configuration
 - Endpoint (endpoint) - [REQ] Endpoint of Octa from which data will be fetched
 - Search (search) - [OPT] Refer to <a href=https://developer.okta.com/docs/reference/api/users/#list-users-with-search>Okta Docs</a> for query syntax.
 - Sync Mode (sync_mode) - [OPT] Full Sync downloads all data from the source every run, Incremental Sync downloads data updated since the last run time.
 - Storage Table Name (output_table_name) - [REQ] Name of the table stored in Storage.
 - Load Type (load_type) - [REQ] If Full load is used, the destination table will be overwritten every run. If incremental load is used, data will be upserted into the destination table. Tables with a primary key will have rows updated, tables without a primary key will have rows appended.
 - Primary Keys (primary_keys) - [OPT] List of primary keys separated by commas e.g. id, other_id. If a primary key is set, updates can be done on the table by selecting incremental loads. The primary key can consist of multiple columns. The primary key of an existing table cannot be changed.


Sample Configuration
=============
```json
{
    "parameters": {
        "credentials": {
            "org_url": "keboola.okta.com",
            "#api_token": "{REDACTED}"
        },
        "endpoint": "USERS",
        "search": "status eq \"STAGED\"",
        "sync_options": {
            "sync_mode": "incremental_sync",
            "date_from": "7 days ago",
            "date_to": "now"
        },
        "destination": {
            "load_type": "incremental_load",
            "output_table_name": "okta_test"
        },
        "debug": false
    },
    "action": "run"
}
```


Development
-----------

If required, change local data folder (the `CUSTOM_FOLDER` placeholder) path to your custom path in
the `docker-compose.yml` file:

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    volumes:
      - ./:/code
      - ./CUSTOM_FOLDER:/data
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Clone this repository, init the workspace and run the component with following command:

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
docker-compose build
docker-compose run --rm dev
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Run the test suite and lint check using this command:

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
docker-compose run --rm test
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

File: deploy.sh
================================================
#!/bin/sh
set -e

env

# compatibility with travis and bitbucket
if [ ! -z ${BITBUCKET_TAG} ]
then
	echo "assigning bitbucket tag"
	export TAG="$BITBUCKET_TAG"
elif [ ! -z ${TRAVIS_TAG} ]
then
	echo "assigning travis tag"
	export TAG="$TRAVIS_TAG"
elif [ ! -z ${GITHUB_TAG} ]
then
	echo "assigning github tag"
	export TAG="$GITHUB_TAG"
else
	echo No Tag is set!
	exit 1
fi

echo "Tag is '${TAG}'"

#check if deployment is triggered only in master
if [ ${BITBUCKET_BRANCH} != "master" ]; then
               echo Deploy on tagged commit can be only executed in master!
               exit 1
fi

# Obtain the component repository and log in
echo "Obtain the component repository and log in"
docker pull quay.io/keboola/developer-portal-cli-v2:latest
export REPOSITORY=`docker run --rm  \
    -e KBC_DEVELOPERPORTAL_USERNAME \
    -e KBC_DEVELOPERPORTAL_PASSWORD \
    quay.io/keboola/developer-portal-cli-v2:latest \
    ecr:get-repository ${KBC_DEVELOPERPORTAL_VENDOR} ${KBC_DEVELOPERPORTAL_APP}`

echo "Set credentials"
eval $(docker run --rm \
    -e KBC_DEVELOPERPORTAL_USERNAME \
    -e KBC_DEVELOPERPORTAL_PASSWORD \
    quay.io/keboola/developer-portal-cli-v2:latest \
    ecr:get-login ${KBC_DEVELOPERPORTAL_VENDOR} ${KBC_DEVELOPERPORTAL_APP})

# Push to the repository
echo "Push to the repository"
docker tag ${APP_IMAGE}:latest ${REPOSITORY}:${TAG}
docker tag ${APP_IMAGE}:latest ${REPOSITORY}:latest
docker push ${REPOSITORY}:${TAG}
docker push ${REPOSITORY}:latest

# Update the tag in Keboola Developer Portal -> Deploy to KBC
if echo ${TAG} | grep -c '^v\?[0-9]\+\.[0-9]\+\.[0-9]\+$'
then
    docker run --rm \
        -e KBC_DEVELOPERPORTAL_USERNAME \
        -e KBC_DEVELOPERPORTAL_PASSWORD \
        quay.io/keboola/developer-portal-cli-v2:latest \
        update-app-repository ${KBC_DEVELOPERPORTAL_VENDOR} ${KBC_DEVELOPERPORTAL_APP} ${TAG} ecr ${REPOSITORY}
else
    echo "Skipping deployment to KBC, tag ${TAG} is not allowed."
fi

import logging
from datetime import datetime, timezone
import os

from keboola.component.base import ComponentBase
from keboola.component.exceptions import UserException
from keboola.csvwriter import ElasticDictWriter
from keboola.utils import parse_datetime_interval

from client.okta_client import OktaClient
from client.mapping import mapping

# configuration variables
KEY_GROUP_CREDENTIALS = 'credentials'
KEY_GROUP_SYNC_OPTIONS = 'sync_options'
KEY_GROUP_DESTINATION = 'destination'

KEY_API_TOKEN = '#api_token'
KEY_ORG_URL = 'org_url'
KEY_ENDPOINT = 'endpoint'
KEY_SYNC_MODE = 'sync_mode'
KEY_SEARCH = 'search'
KEY_LOAD_TYPE = 'full_load'
KEY_OUTPUT_TABLE_NAME = 'output_table_name'
KEY_DATE_FROM = 'date_from'
KEY_DATE_TO = 'date_to'

# list of mandatory parameters => if some is missing,
# component will fail with readable message on initialization.
REQUIRED_PARAMETERS = [KEY_GROUP_CREDENTIALS, KEY_ENDPOINT]

primary_key = ["id"]


class Component(ComponentBase):

    def __init__(self):
        super().__init__()
        self.processed_rows = 0

    def run(self):
        self.validate_configuration_parameters(REQUIRED_PARAMETERS)
        params = self.configuration.parameters

        credentials = self.configuration.parameters.get(KEY_GROUP_CREDENTIALS)
        sync_options = self.configuration.parameters.get(KEY_GROUP_SYNC_OPTIONS)
        destination = self.configuration.parameters.get(KEY_GROUP_DESTINATION)

        current_time = datetime.now(timezone.utc).strftime('%Y-%m-%dT%H:%M:%S.%f')[:-3] + 'Z'

        token = credentials.get(KEY_API_TOKEN)
        org_url = credentials.get(KEY_ORG_URL)
        endpoint = params.get(KEY_ENDPOINT).lower()
        statefile = self.get_state_file()

        date_from, date_to = self.handle_sync_mode(sync_options, statefile, current_time)
        incremental_load = self.handle_load_type(destination)

        if date_from:
            self.validate_datetime_strings(date_from)
        if date_to:
            self.validate_datetime_strings(date_to)

        # Search
        if params.get(KEY_SEARCH, {}):
            if mapping.get(endpoint).get("supports_search", False):
                search = params.get(KEY_SEARCH)
            else:
                logging.warning(f"Endpoint {endpoint} does not support search function.")
                search = False
        else:
            search = False

        # Fieldnames
        if statefile.get("fieldnames", {}):
            fieldnames = statefile.get("fieldnames")
        else:
            fieldnames = []

        # Table name
        if destination.get(KEY_OUTPUT_TABLE_NAME, {}):
            output_table_name = destination.get(KEY_OUTPUT_TABLE_NAME)
            logging.info(f"Using custom table name: {output_table_name}")
        else:
            output_table_name = endpoint

        client = OktaClient(org_url, token)

        table_def = self.create_out_table_definition(name=output_table_name, incremental=incremental_load,
                                                     primary_key=primary_key)

        with ElasticDictWriter(table_def.full_path, fieldnames) as wr:
            wr.writeheader()
            for record in client.get_records(endpoint, date_from, date_to, search):
                wr.writerow(record)
                self.processed_rows += 1

        if self.processed_rows >= 1:
            self.write_manifest(table_def)
        else:
            os.remove(table_def.full_path)

        state_to_write = {
            "last_run_time": current_time,
            "columns": wr.fieldnames
        }
        self.write_state_file(state_to_write)
        logging.info(f"Saved run_time to statefile: {current_time}")
        logging.info(f"Processed rows: {self.processed_rows}")

    @staticmethod
    def validate_datetime_strings(s):
        format_str = '%Y-%m-%dT%H:%M:%S.%fZ'
        try:
            datetime.strptime(s, format_str)
            return
        except ValueError:
            raise UserException(f"Datetime string {s} cannot be parsed in a {format_str} format.")

    @staticmethod
    def get_date_from(sync_options, statefile, parser_dt_format):
        date_from_option = sync_options.get(KEY_DATE_FROM, "last run")

        if date_from_option == "last run":
            last_run_time = statefile.get("last_run_time")
            if last_run_time:
                logging.info("Sync mode: Incremental")
                logging.info(f"Loaded last_run_time from statefile: {last_run_time}")
                return last_run_time
            else:
                logging.info("last_run_time not found in statefile. Component will do full sync.")
                return False

        try:
            date_from, _ = parse_datetime_interval(date_from_option, "today", parser_dt_format)
        except TypeError as e:
            raise UserException(f"Cannot parse {date_from_option} as date.") from e
        return str(date_from) + "T00:00:00.000Z"

    @staticmethod
    def get_date_to(sync_options, current_time, parser_dt_format):
        date_to_option = sync_options.get(KEY_DATE_TO, False)

        if date_to_option == "now":
            return current_time
        elif date_to_option:
            try:
                date_to, _ = parse_datetime_interval(date_to_option, "today", parser_dt_format)
            except TypeError as e:
                raise UserException(f"Cannot parse {date_to_option} as date.") from e
            return str(date_to) + "T00:00:00.000Z"
        return False

    def handle_sync_mode(self, sync_options, statefile, current_time):
        sync_mode = sync_options.get(KEY_SYNC_MODE, {})
        parser_dt_format = "%Y-%m-%d"
        date_from = date_to = False

        if sync_mode == "incremental_sync":
            date_from = self.get_date_from(sync_options, statefile, parser_dt_format)
            date_to = self.get_date_to(sync_options, current_time, parser_dt_format)
        else:
            logging.info("Sync mode: Full Sync")

        return date_from, date_to

    @staticmethod
    def handle_load_type(destination):
        load_type = destination.get(KEY_LOAD_TYPE, {})
        if load_type == "incremental_load":
            logging.info("Load Type: Incremental")
            return True
        else:
            logging.info("Load Type: Full")
            return False

from keboola.http_client import HttpClient

from requests.exceptions import HTTPError
import logging
import collections.abc
import backoff
import time

from .mapping import mapping


class OktaClientError(Exception):
    pass


def flatten_dict(d, parent_key='', sep='_'):
    """
    This function flattens a dictionary of dictionaries into a single level dictionary.
    """
    items = []
    for k, v in d.items():
        new_key = parent_key + sep + k if parent_key else k
        if isinstance(v, collections.abc.MutableMapping):
            items.extend(flatten_dict(v, new_key, sep=sep).items())
        else:
            items.append((new_key, v))
    return dict(items)


def get_next_url_from_headers(headers):
    link_header = headers.get('link')
    if link_header:
        links = link_header.split(',')
        for link in links:
            parts = link.split(';')
            if len(parts) == 2 and 'rel="next"' in parts[1]:
                url = parts[0].strip()[1:-1]  # remove angle brackets around URL
                return url
    return None


class OktaClient:
    supported_endpoints = ["users", "user_types", "devices"]

    def __init__(self, org_url, token):

        self.date_to = None
        self.date_from = None
        self.base_url = f"https://{org_url}/api/v1"

        default_header = {
            'Accept': 'application/json',
            'Content-Type': 'application/json',
            'Authorization': f'SSWS {token}'
        }

        self.limit = 250
        self.client = HttpClient(self.base_url, default_http_header=default_header)

    def get_records(self, endpoint, date_from, date_to, search=False):

        self.date_from = date_from
        self.date_to = date_to

        if endpoint not in self.supported_endpoints:
            raise OktaClientError(f"{endpoint} is not a supported endpoint. "
                                  f"Supported endpoints: {self.supported_endpoints}")

        try:
            url = mapping.get(endpoint).get("url")
        except AttributeError:
            raise OktaClientError(f"Endpoint {endpoint} is not supported.")

        logging.info(f"Processing endpoint: {endpoint}")
        for page in self.fetch(url, search):
            for record in page:
                yield flatten_dict(record)

    def build_search_params(self, search):
        params = {}

        if search:
            params = {"search": search}
            logging.info(f"Search enabled: {params}")

        search_filter = ""
        if self.date_from:
            if search:
                search_filter += " and "
            search_filter += f'lastUpdated gt "{self.date_from}"'
        if self.date_to:
            if search_filter or search:
                search_filter += " and "
            search_filter += f'lastUpdated lt "{self.date_to}"'

        if search_filter:
            params["search"] = str(params.get("search", "")) + str(search_filter)

        return params

    @backoff.on_exception(backoff.expo, HTTPError, max_time=60)
    def fetch(self, url, search=False, is_absolute=False) -> dict:
        while True:
            params = self.build_search_params(search) if not is_absolute else {}

            if not is_absolute:
                params["limit"] = self.limit

            r = self.client.get_raw(url, is_absolute_path=is_absolute, params=params)

            try:
                r.raise_for_status()
            except HTTPError as e:
                if e.response.status_code == 429:
                    retry_after = int(e.response.headers.get("Retry-After", 30))
                    logging.warning(f"Received 429 status_code, sleeping for {retry_after} seconds before retrying.")
                    time.sleep(retry_after)
                    continue
                else:
                    raise

            yield r.json()

            next_url = get_next_url_from_headers(r.headers)
            if next_url:
                url = next_url
                is_absolute = True
            else:
                break
